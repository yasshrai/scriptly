# Install IntelliJ IDEA

The most popular IDE for Java development, now available via Scriptly.

## Install via cURL

Run the following command in your terminal. It will download and execute the installation script directly:

```bash
curl -sL {{BASE_URL}}/scripts/installintellijidea.sh | sudo bash
```

## Update IntelliJ IDEA

To update to the latest version, simply re-run the installation command above. The script will safely replace your existing installation.

## Uninstall IntelliJ IDEA

### Uninstall via cURL

```bash
curl -sL {{BASE_URL}}/scripts/uninstall/uninstallintellijidea.sh | sudo bash
```
