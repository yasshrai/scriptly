# Install VS Code

Reliable installations from official sources for Linux x64.

## Install via cURL

Run the following command in your terminal. It will download and execute the installation script directly:

```bash
curl -sL {{BASE_URL}}/scripts/installvscode.sh | sudo bash
```

## Update VS Code

To update to the latest version, simply re-run the installation command above. The script will safely replace your existing installation.

## Uninstall VS Code

### Uninstall via cURL

```bash
curl -sL {{BASE_URL}}/scripts/uninstall/uninstallvscode.sh | sudo bash
```
