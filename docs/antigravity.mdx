# Install Antigravity

The agentic AI coding assistant, now available via Scriptly.

## Install via cURL

Run the following command in your terminal. It will download and execute the installation script directly:

```bash
curl -sL {{BASE_URL}}/scripts/installantigravity.sh | sudo bash
```

## Update Antigravity

To update to the latest version, simply re-run the installation command above. The script will safely replace your existing installation.

## Uninstall Antigravity

### Uninstall via cURL

```bash
curl -sL {{BASE_URL}}/scripts/uninstall/uninstallantigravity.sh | sudo bash
```
