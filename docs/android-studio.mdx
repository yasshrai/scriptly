# Install Android Studio

Build amazing Android apps with a clean installation script.

## Install via cURL

Run the following command in your terminal. It will download and execute the installation script directly:

```bash
curl -sL {{BASE_URL}}/scripts/installandroidstudio.sh | sudo bash
```

## Update Android Studio

To update to the latest version, simply re-run the installation command above. The script will safely replace your existing installation.

## Uninstall Android Studio

### Uninstall via cURL

```bash
curl -sL {{BASE_URL}}/scripts/uninstall/uninstallandroidstudio.sh | sudo bash
```
