# Install Antigravity

The agentic AI coding assistant, now available via Scriptly.

### 1. Clone the repository
```bash
git clone https://github.com/yasshrai/scriptly
```

### 2. Navigate to scripts
```bash
cd scriptly/scripts
```

### 3. Make the script executable
```bash
chmod +x installantigravity.sh
```

### 4. Run the installer
```bash
sudo ./installantigravity.sh
```
