# Install Windsurf

The innovative AI integrated IDE, now available via Scriptly.

## Install via cURL

Run the following command in your terminal. It will download and execute the installation script directly:

```bash
curl -sL {{BASE_URL}}/scripts/installwindsurf.sh | sudo bash
```

## Update Windsurf

To update to the latest version, simply re-run the installation command above. The script will safely replace your existing installation.

## Uninstall Windsurf

### Uninstall via cURL

```bash
curl -sL {{BASE_URL}}/scripts/uninstall/uninstallwindsurf.sh | sudo bash
```
