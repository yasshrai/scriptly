# Install Cursor

The AI-powered code editor, now available via Scriptly.

### Install via cURL

Run the following command in your terminal. It will download and execute the installation script directly:

```bash
curl -sL {{BASE_URL}}/scripts/installcursor.sh | sudo bash
```
# Uninstall Cursor

### Uninstall via cURL

```bash
curl -sL {{BASE_URL}}/scripts/uninstall/uninstallcursor.sh | sudo bash
```
