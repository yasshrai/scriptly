# Install PyCharm

Professional Python development on Linux, simplified.

## Install via cURL

Run the following command in your terminal. It will download and execute the installation script directly:

```bash
curl -sL {{BASE_URL}}/scripts/installpycharm.sh | sudo bash
```

## Update PyCharm

To update to the latest version, simply re-run the installation command above. The script will safely replace your existing installation.

## Uninstall PyCharm

### Uninstall via cURL

```bash
curl -sL {{BASE_URL}}/scripts/uninstall/uninstallpycharm.sh | sudo bash
```
